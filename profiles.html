Profils PCM – Conseils de communication

🧠
Penseur·se

  • À faire : être structuré·e, logique, précis·e
  • Utilise : faits, procédures, explications claires
  • Évite : l'imprécision, l'improvisation, les débordements émotionnels

❤️
Empathique

  • À faire : être chaleureux·se, humain·e, à l'écoute
  • Utilise : reconnaissance, émotions, attention à l'autre
  • Évite : le ton froid, distant ou impersonnel

🌟
Rebelle

  • À faire : être léger·e, spontané·e, ludique
  • Utilise : humour, créativité, interaction dynamique
  • Évite : la rigidité, les règles trop strictes

📢
Promoteur·rice

  • À faire : être direct·e, rapide, axé·e sur les résultats
  • Utilise : défis, autonomie, action
  • Évite : les détails excessifs, les lenteurs, le contrôle excessif

🧘
Rêveur·se

  • À faire : donner de l'espace, être calme et structuré·e
  • Utilise : consignes claires, environnement paisible
  • Évite : la précipitation, la surcharge de stimulations

🧭
Persévérant·e

  • À faire : être cohérent·e, engagé·e, respectueux·se des convictions
  • Utilise : valeurs, justice, reconnaissance de l'engagement
  • Évite : l'incohérence, le désintérêt apparent
